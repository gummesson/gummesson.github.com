About

My name is Ellen Gummesson and I love communication and the web. I recently recieved a degree in media and communication studies, specializing in strategic communication. Lately I’ve been diving straight into front-end development, mainly writing HTML5 and CSS3, dabbling a little bit with jQuery and starting to really enjoy JavaScript. I also like Sass and love Compass.

I’ve also started to learn the programming language Ruby and I’m hoping that one day I’ll conquer the Ruby on Rails framework. In the meantime I’m having fun with micro web framework Sinatra. I enjoy writing small but practically useless shell scripts that I run on my Debian Testing box. You can check out my projects on Gitub if you’d like, although I’m still very much a beginner when it comes to coding.

Colophon

This site is powered by Jekyll and hosted on GitHub Pages. It was built with HTML5, CSS3 and Sass. The code was written by hand in Vim. The site makes use of the rather nice looking typeface Lato, which is provided by Google Webfonts.

The blog posts are written in Markdown, scribbled down in Pop, iA Writer or ResophNotes before finally ending up and being finished in FocusWriter, Vim or occasionally in MarkdownPad. Shotty is used for taking the screenshots.
