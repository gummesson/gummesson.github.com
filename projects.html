Projects

Down below you’ll find my humble collection of apps, projects, demos and snippets written Ruby, PHP, HTML and CSS, a couple of handy AutoHotKey scripts and some themes for the app FocusWriter.

Kramit!

Ruby/Sinatra

Kramit! is a Markdown preview app built with Sinatra and kramdown. It’s deployed with and hosted on Heroku.

Try it / Download / GitHub

Slugize

Ruby/Sinatra

Slugize is a small web app built with Sinatra that transforms a blog post title or any other kind of string into a slug.

Try it / Download / GitHub

Journal

Ruby/Sinatra

Journal is a simple journal/diary app built with Sinatra which uses DataMapper along with SQLite3 for storing everything in a database. It also uses Rack::Basic:Auth for basic user authenication.

Download / GitHub

Serve

Ruby

Serve is probably the ugliest static site generator in the world but hey, it gets the job done. It’s also my first real attempt at writing a more complete and useful Ruby “app”.

Download / GitHub

Render

Ruby

Render is a Mardown parser/preview tool. The default parser is the kramdown gem, but it can also use Redcarpet, RDiscount or Maruku.

Download / GitHub

Jekyll Rake Boilerplate

Ruby

Jekyll Rake Boilerplate is a small rake “boilerplate” for doing common tasks with the static site generator Jekyll, such as generating your site (with an optional post limit), creating a new post from a default template and deploying it to it’s remote git repository.

Download / GitHub

Jekyll Vimeo Plugin

Ruby

Jekyll Vimeo Plugin is a plugin for Jekyll that makes it easy to embed videos from Vimeo by using a simple Liquid tag. It’s based of the Youtube plugin/gist from Portway Point.

Download / GitHub

Smallwork

CSS/SCSS

Smallwork is a small and extendable SCSS framework which consists of mixins, variables and partials that serves as a good starting point for various projects.

Download / GitHub

Gridy

CSS/SCSS

Gridy is a small SCSS grid framework. It can be used both as base for a class-based grid system or applied directly on the markup.

Download / GitHub

Clean Slate For GetSimple CMS

CSS/HTML/PHP

Clean Slate is a bare bones starter theme for GetSimple CMS. It contains some minimal markup, a basic reset and a couple of OOCSS-inspired classes to get you started, but that’s it.

Download / GitHub

Markdown Hostrings (for AutoHotKey)

AutoHotKey

Markdown Hostrings is a collection of AutoHotKey scripts that helps you to write Markdown even more faster. It also includes hotstrings for footnotes and comments.

Download / GitHub

Markdown Keybindings (for AutoHotKey)

AutoHotKey

Markdown Keybindings is a small collection of keybindings for streamlining your Markdown writing.

Download / GitHub

FocusWriter Themes

FocusWriter Theme Files

FocusWriter Themes is a collection of a couple of themes I created for the excellent free text editor FocusWriter. They’re based on some popular color schemes from various other editors.
