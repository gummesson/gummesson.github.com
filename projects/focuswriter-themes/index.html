FocusWriter Themes

Here’s a couple of themes I created for the excellent free text editor FocusWriter. They’re based on some popular color schemes from various other editors.

Typeface

The themes use the font DejaVu Sans Mono, which is available here. It also comes pre-installed with Windows and a couple of Linux distros. If you’re using OSX and don’t feel like downloading it, Menlo is a great alternative.

iA Writer

iA Writer is based on iA Writer from Information Architects.

Background (RGB): 242, 242, 244
Text (RGB): 91, 91, 93

Byword

Byword is based on the dark theme in Byword from Metaclassy.

Background (RGB): 26, 26, 24
Text (RGB): 191, 191, 193

Cobalt

Cobalt is based on the Cobalt theme from Jacob Rus.

Background (RGB): 0, 34, 64
Text (RGB): 58, 217, 56

Tomorrow Night Eighties

Tomorrow Night Eighties is based on the Tomorrow Night Eighties theme from Chris Kempson.

Background (RGB): 45, 45, 45
Text (RGB): 242, 119, 122

Solarized

The Solarized themes are based on the Solarized theme from Ethan Schoonover.

Solarized Dark Mellow

Background (RGB): 0, 43, 54
Text (RGB): 101, 123, 131

Solarized Dark Bright

Background (RGB): 0, 43, 54
Text (RGB): 42, 161, 152

Solarized Light Mellow

Background (RGB): 253, 246, 227
Text (RGB): 101, 123, 131

Solarized Light Bright

Background (RGB): 253, 246, 227
Text (RGB): 42, 161, 152

Pocket

The Pocket theme is based of one of color schemes from the iOS app Pocket.

Background (RGB): 251, 240, 217
Text (RGB): 91, 70, 54
